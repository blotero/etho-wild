# Welcome to EthoWild

**EthoWild** is a desktop application for labeling animal behaviors in video recordings. Built with C++ and Qt6, it provides a fast, intuitive interface for researchers studying wildlife behavior.

## Features

- **Video Playback** - Play, pause, seek, and adjust playback speed (0.5x to 2.0x)
- **Directory Navigation** - Load an entire folder of videos and navigate between them
- **Zoom & Pan** - Ctrl+scroll to zoom, drag to pan for detailed observation
- **Behavior Labeling** - Record events (instant) and states (duration-based) with a single double-click
- **Session Metadata** - Tag, role, sex, stage, group type, group size, and observations
- **CSV Export** - Export all labeled records to CSV for analysis

## Quick Start

1. [Download](installation.md#download) the latest release for your platform
2. Place `behaviors.json` in the same directory as the executable
3. Launch EthoWild and open a video or video directory
4. Start labeling behaviors!
